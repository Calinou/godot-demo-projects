# 2D Skeletons, Polygons and Lines

Simple demo of 2D skeletons, polygons and lines, using
[`Skeleton2D`](https://docs.godotengine.org/en/latest/classes/class_skeleton2d.html),
[`Polygon2D`](https://docs.godotengine.org/en/latest/classes/class_polygon2d.html), and
[`Line2D`](https://docs.godotengine.org/en/latest/classes/class_line2d.html).

Language: GDScript

Renderer: GLES 2
